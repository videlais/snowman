## Styling Text

Snowman supports most (but not all) of the syntax possibilities provided by the Marked library. Any multiple-line syntax, notably variants on creating headers, is not supported. All input is rendered on a per-line basis.

## Headers

| Syntax   | Code              |
|----------|-------------------|
| Header 1 | `# Header 1`      |
| Header 2 | `## Header 2`    |
| Header 3 | `### Header 3`    |
| Header 4 | `#### Header 4`   |
| Header 5 | `##### Header 5`  |
| Header 6 | `###### Header 6` |

## Styles

| Syntax   | Code         |
|----------|--------------|
| Strong   | `**Strong**` |
| Emphasis | `*Emphasis*` |

## Line Items

| Syntax           | Code             |
|------------------|------------------|
| Unordered Item   | `- Item`         |
| Unordered Item   | `* Item`         |
| Ordered Item     | `1. Item`        |
