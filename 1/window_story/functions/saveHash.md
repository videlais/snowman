## `saveHash()`

`saveHash()` is a function of the `window.story` global JavaScript variable created by Snowman.

*Name*: saveHash()

*Parameters*: (None.)

*Return Type*: {String} Hash

*Description*: Returns a hash value representing the current state of the story.
