## `save()`

`save()` is a function of the `window.story` global JavaScript variable created by Snowman.

*Name*: save()

*Parameters*: (None.)

*Return Type*: {String} Hash

*Description*: Sets the URL's hash property to the hash value created by `saveHash()`.
